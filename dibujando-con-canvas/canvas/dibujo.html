Dibujando en Canvas

Ya qué sabemos lo que es el DOM y como manipularlo como objeto, vamos a hacer un dibujo usando la etiqueta "canvas" de Html. Es decir, vamos a dibujar dentro del DOM.

Maneras diferentes para aplicar el manejo del DOM y los dibujos

Manera sencilla

Tenemos la primera y más simple forma de aplicar lo aprendido. Un dibujo, sea complejo o llamativamente simple (como el siguiente) puede hacerse al manipular las propiedades que tenemos en el elemento canvas. Se pueden hacer dibujos 3d o 2d y meter lineas adentro de una manera particular para plasmar nuestras más crativas creaciones.

Este dibujo, aunque no lo parezca, está solamente creado con lineas rectas. Estas lineas rectas fueron dibujadas por una función en JavaScript y un ciclo en el que cada linea se multiplica y todas estas se distribuyen a travez del canvas

Interactividad

Sí queremos interactividad, es decir, que el usuario determine el color y la posicion de las lineas, esto también es posible. Vamos a aumentar el codigo en JavaScript, añadir botones e inputs para que el usuario pueda decirle al codigo que es lo que quiere y personalizar sus dibujos de una manera instantanea.
