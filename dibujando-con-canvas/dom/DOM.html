Jugando con el DOM

Resulta que dentro del navegador existen objetos importantes que tu puedes manipular a travez de Java Script.

  • Navigator: Navigator es simplemente todo el navegador, y con este puedes manipular cosas del propio navegador e incluso del Sistema Operativo
  • Window: Window es cada una de las ventanas o "tabs" que se encuentran abiertas en el navegador, por ejemplo ahí tienes la URL.
  • Document: Document es todo el documento en HTML de la página que estás construyendo. Ahí se encuentran todo el arbol de elementos HTML.
Un objeto puede tener métodos y atributos o valores. Los métodos son las funciones que tiene el objeto. Ej: document.write();
Write es una función que va a coger el valor que tu colocaste detro del parentesis y lo va a escribir dentro del objeto documento, es decir, dentro del HTML.
Los atrubutos son valores, características del objeto, no son ejecutables como las funciones (metodos). Ej: window.location. El valor que te devuelve es la URL de la ventana en la que te encuentres, ya es un valor guardado en memoria, no una función.

Cuando quieres depurar un codigo, tienes la opcion de usar la console. Depurar es el proceso de tratar de arreglar un error en el codigo.

console.log("Vamos a arreglar un error");
Se puede usar porque solo los desarrolladores a travez de la consola del navegador pueden ver ese mensaje.
Tambien se usa para mirar objetos junto con sus metodos y atributos.
console.log(window.location);
Esto te abrirá el objeto window.location dentro de la consola y podrás ver todo lo que necesitas. Esto crea algo curioso, puesto que tambien puedes ver el objeto 'console' y el metodo 'console.log()' desde la consola.
